1

HTML5

HTML5 é o esqueleto da web. Ele estrutura o conteúdo (textos, imagens, links) que vemos nas páginas.

2

CSS3

CSS é a maquiagem da web. Ele estiliza o HTML, controlando cores, fontes, layouts e o posicionamento dos elementos.

3

JavaScript

JavaScript (JS) é o músculo da web. Ele adiciona interatividade e lógica, fazendo as páginas dinâmicas e funcionais.

4

Python

Python é uma linguagem versátil, muito usada no Backend (servidor), análise de dados e inteligência artificial, sendo famosa por sua sintaxe clara.

5

Node.js

Node.js permite usar JavaScript no **Backend** (fora do navegador). É usado para construir APIs rápidas e servidores escaláveis.

6

React

React é uma biblioteca JavaScript para criar **Interfaces de Usuário (UI)** interativas e complexas, baseada no conceito de componentes reutilizáveis.

7

TypeScript

TypeScript é um superset do JavaScript que adiciona tipagem estática. Ele ajuda a evitar erros e deixa o código mais organizado e escalável.

8

Git & GitHub

Git é um sistema de controle de versão e o GitHub é a plataforma onde você armazena e compartilha seus projetos. Essenciais para trabalhar em equipe.

9

SQL

SQL é a linguagem usada para criar, consultar e gerenciar bancos de dados. Fundamental para qualquer aplicação que precise armazenar informações.
